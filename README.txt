Pour compiler le programme, si l'ensemble des fichiers se trouvent dans le même répertoire:
javac Main.java

Pour executer le programme:
java Main

Le fichier fourni doit être sous le format accepté par la librairie Graph de algs4.jar.
Exemple de fichiers:
https://algs4.cs.princeton.edu/41graph/tinyG.txt
https://algs4.cs.princeton.edu/41graph/mediumG.txt
https://algs4.cs.princeton.edu/41graph/largeG.txt

Les fichier de codes qui ne sont pas issus de la librairie algs4.jar et par conséquent coder dans le cadre de ce cours sont:
- Degenerescence.java
- ChromaNumber.java
- Main.java

Nous avons également le fichier Denerancy Algorithme qui contient un algorithme fonctionnel mais en un temps beaucoup moins efficace. 
Ce bout de code n'est pas appelé par Main.
